data盘挂载

    ① 查看系统中所有磁盘设备：

         sudo fdisk -l

    ② 对磁盘进行分区：

         sudo fdisk /dev/sdb # 将sdb替换成实际磁盘名称（输入n创建新分区；输入w保存分区表；按回车键采用默认设置）

    ③ 格式化分区：

         sudo mkfs.ext4 /dev/sdb1 # 假设分区为sdb1

    ④ 创建挂载点：

         sudo mkdir /data # 创建目录“data”作为数据盘分区“sdb1”的挂载位置

    ⑤ 挂载磁盘：

         sudo mount /dev/sdb1 /data # 将数据盘分区“sdb1”挂载到目录“data”

    ⑥ 永久挂载（系统重启后仍保持挂载）：

         sudo blkid /dev/sdb1 # 获取数据盘分区“sdb1”的UUID

         sudo nano /etc/fstab # 编辑/etc/fstab文件

         UUID=3e6be9de-8139-4104-8de9-4b2b64d64b2d /data ext4 defaults 0 0 # 在文件末尾添加一行（换成刚刚获取的UUID）；Ctrl + O 保存； Ctrl + X 退出

    ⑦ 验证：

         df -h

Zotero

    ① 安装教程：

         （1）Zotero下载链接https://www.zotero.org/support/dev/zotero_7_for_developers

         （2）tar -xjvf Zotero-7.0.15_linux-x86_64.tar.bz2 -C ~/ # 解压并将其放在“home”目录

         （3）nano ~/.local/share/applications/zotero.desktop # 创建一个.desktop文件，并复制下面的内容：

         [Desktop Entry]

         Name=Zotero

         Comment=Research tool

         Exec=/home/xxx/Zotero/zotero # xxx替换为实际名字

         Icon=/home/xxx/Zotero/icons/icon128.png # 这里替换为实际的桌面图标路径

         Terminal=false

         Type=Application

         Categories=Office;

         StartupNotify=true

         （4）依次按：Ctrl + X → Y → Enter

    ② 一些注意事项：

         （1）详细的安装教程可以参考这篇博客：https://developer.baidu.com/article/details/2747033

         （2）文档同步：Zotero文件夹下的storages目录中存放着所有的文献，可以直接复制到新主机的目录下，然后进行同步

         （3）Linux系统无法下载小绿鲸

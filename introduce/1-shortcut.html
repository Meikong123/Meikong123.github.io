虚拟环境

    ① 创建一个名为 "sam" ，Python版本为 ” 3.9.19“ 的虚拟环境：

         conda create -n sam python=3.9.19

    ② 激活名为 “sam” 的虚拟环境：

         conda activate sam

    ③ 查看所有的虚拟环境：

         conda env list或conda info -e或conda info --env

    ④ 退出当前的虚拟环境：

         conda deactivate

    ⑤ 移除一个名为 "sam" 的虚拟环境：

         conda remove --name sam --all

    ⑥ 克隆一个名为"sam1"的虚拟环境，并命名为"sam2"：

         conda create --name sam2 --clone sam1

GPU

    ① 查看 GPU 信息：

         （简略）nvidia-smi     （详细）nvidia-smi -q

    ② 使用指定的某些（2,3）张卡：

         export CUDA_VISIBLE_DEVICES=2,3

Cmd上运行Python程序

    ① 导航到目标地址“/data/xxx/projects” 路径下：

         cd /data/xxx/projects

    ② 运行名为 “1.py" 的Python程序（先导航到目录）:

         python 1.py
